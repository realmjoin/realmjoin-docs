# Get Bitlocker Recovery Key

Get BitLocker recovery key

## Description

Get BitLocker recovery key via supplying bitlockeryRecoveryKeyId.

## Location

Org &rarr; Devices &rarr; Get Bitlocker Recovery Key

## Permissions

### API Permissions

**Microsoft Graph**
- Device.Read.All
- BitlockerKey.Read.All

## Parameters

### -CallerName

**Description:** Caller name for auditing purposes. 

| Name | Value |
|---|---|

| Position | _1_ |
