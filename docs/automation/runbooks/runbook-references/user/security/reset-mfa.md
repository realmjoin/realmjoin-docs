## Description
Remove all App- and Mobilephone auth methods for a user. User can re-enroll MFA.

## Location
User → Security → Reset Mfa

## Permissions
### Application permissions
- **Type**: Microsoft Graph
  - UserAuthenticationMethod.ReadWrite.All


## Parameters
### UserName



| Property | Value |
| --- | --- |
| Required | true |
| Default Value |  |
| Type | String |
