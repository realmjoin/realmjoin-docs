## Description
Assign a given OWA mailbox policy to a user. E.g. to allow MS Bookings.

## Location
User → Mail → Assign Owa Mailbox Policy

## Permissions
### RBAC roles
- Exchange administrator


## Parameters
### UserName



| Property | Value |
| --- | --- |
| Required | true |
| Default Value |  |
| Type | String |

### OwaPolicyName



| Property | Value |
| --- | --- |
| Required | true |
| Default Value | OwaMailboxPolicy-Default |
| Type | String |
