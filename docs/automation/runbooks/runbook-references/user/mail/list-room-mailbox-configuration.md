## Description
Reads room metadata and lists calendar processing settings. This helps validate room resource configuration and booking behavior.

## Location
User → Mail → List Room Mailbox Configuration

## Permissions
### Application permissions
- **Type**: MG Graph
  - Place.Read.All


## Parameters
