# Reset Mfa

Remove all App- and Mobilephone auth methods for a user.

## Description

Remove all App- and Mobilephone auth methods for a user. User can re-enroll MFA.

## Location

## Notes

Permissions needed:
- UserAuthenticationMethod.ReadWrite.All
