# Rename Group

Rename a group.

## Description

Rename a group MailNickname, DisplayName and Description. Will NOT change eMail addresses!

## Location

Group &rarr; General &rarr; Rename Group

## Permissions

### API Permissions

**Microsoft Graph**
- Group.ReadWrite.All
