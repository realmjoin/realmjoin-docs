## Description
Disable a device in AzureAD.

## Location
Device → Security → Enable Or Disable Device

## Permissions
### Application permissions
- **Type**: Microsoft Graph
  - Device.Read.All

### RBAC roles
- Cloud device administrator

| Property | Value |
| --- | --- |
| Required | true |
| Default Value |  |

| Property | Value |
| --- | --- |
| Required | false |
| Default Value | False |
